1. Expense Tracker
   Simple expense tracker is built using Python. It help the user to track every process that is part of their daily expenses.

2. Features
- It takes user input to perform operation on expenses.
- Performs the operation accordingly such as :
  . Check Expenses
  . Amount Deposition
  . Amount Withdrawl
  . View Total Spending
  . Exit
- Users can keep their records in.
- Help them to check later.
